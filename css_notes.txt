Units in CSS
------------
Units are to specify the size of the container
There are 2 types:

1) Absolute Units
    px, in, cm, mm --> FIxed Units

2) Relative Units
    %
    vh --> viewport-height
    vw --> viewport-width
    em --> element-size
    rem --> root-element-size
    calc()-function

    default font size: 16px in html
